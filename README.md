# Exercices sur la correction des capteurs

## 🏗️ Structure d'un exercice

Chaque exercice contient :

- `README.md` : Énoncé de l'exercice
- `<nom_classe>.py` : Fichier à compléter par l'étudiant
- `test_<nom_classe>.py` : Fichier de tests unitaires à compléter par l'étudiant
- `fake_<capteur>.py` : Fichier fourni par l'enseignant pour simuler le capteur (ne pas modifier)

## 📝 Liste des exercices

| Exercice               | Concepts                                                  |
| ---------------------- | --------------------------------------------------------- |
| ex1_offset             | Correction d'une erreur d'offset                          |
| ex2_moyenne_mobile     | Filtrage du bruit de mesure à l'aide d'une moyenne mobile |
| ex3_filtrage           | Filtrage des mesures hors limite                          |
| ex4_calibration        | Calibration d'un potentiomètre virtuel                    |
| ex5_bonus_nonlinearite | Correction de la non-linéarité d'un capteur               |
